## Setup and Run Instructions
### Prerequisites
- Java Development Kit (JDK) 21 or later installed
- Apache Maven installed

### Build and Run
1. Open a terminal and navigate to the root folder of the project (where the `pom.xml` file is located).
2. To compile and build the project, run:
   ```bash
   mvn clean install
3. To start the application, run:
   ```bash
   mvn exec:java -Dexec.mainClass="Main"
